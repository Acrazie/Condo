# Condo
Condo est une plateforme en ligne destinée aux petites copropriétés et aux syndics bénévoles. Elle centralise les tâches, documents, rappels, incidents et décisions afin d’éviter les oublis. L’objectif est de proposer un outil pratique, précis et simple d’utilisation pour gérer efficacement la vie quotidienne d’une copropriété.
